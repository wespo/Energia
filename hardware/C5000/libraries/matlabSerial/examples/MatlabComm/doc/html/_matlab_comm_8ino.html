void loop ( )

Main application loop.

  • Receives two serial commands from Matlab
    • Commands can set the gain or offset
    • 0: sets the gain
    • 1: sets the offset
  • Receives input vector from Matlab
  • Sends output vector to Matlab.
    • Output = gain * input + offset
